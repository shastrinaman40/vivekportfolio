# Vivek Pujari Portfolio Website

A simple, responsive portfolio website for Vivek Pujari, a graphic designer specializing in social media creatives, branding, and visual storytelling.

## Features

- Responsive design that works on desktop and mobile
- Smooth scrolling navigation
- Sections for About, Skills, Portfolio, and Contact
- Clean, modern styling

## Files

- `index.html` - Main HTML structure
- `style.css` - CSS styling
- `script.js` - JavaScript for interactivity
- `placeholder.jpg` - Placeholder image for portfolio projects (replace with actual project images)

## How to View

1. Open `index.html` in a web browser
2. Or serve with a local server for better experience

## Customization

- Replace placeholder images in the portfolio section with actual project screenshots
- Update the content in `index.html` with real project descriptions
- Modify colors and fonts in `style.css` to match personal branding

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
