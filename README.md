# Vivek Pujari Portfolio Website

A modern, responsive portfolio website for Vivek Pujari, a graphic designer specializing in social media creatives, branding, and visual storytelling.

## Features

- Responsive design that works on desktop and mobile
- Smooth scrolling navigation
- Interactive hover effects and animations
- Sections for About, Skills, Portfolio, and Contact
- Clean, modern styling with gradient backgrounds
- Typing animation for hero text

## How to View

### Option 1: Open Locally
1. Download all files to your computer
2. Double-click on `vivek pujari.html` to open in your browser

### Option 2: Use a Local Server
For the best experience with all features working:
1. Install Python (if not already installed)
2. Open command prompt/terminal in the portfolio folder
3. Run: `python -m http.server 8000`
4. Open `http://localhost:8000/vivek%20pujari.html` in your browser

### Option 3: Host Online
Upload all files to any web hosting service (GitHub Pages, Netlify, etc.) and the website will work perfectly.
